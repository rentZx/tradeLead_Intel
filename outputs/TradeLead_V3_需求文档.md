# TradeLead Intel V3.0 需求文档

> 面向发展中国家的外贸获客工具 · 简单 · 免费 · 全自动

---

## 一、产品定位

**一句话**：输入产品，选目标市场，自动搜出潜在客户列表（带背调），一键导出 Excel + 生成多语言开发信。

**核心原则**：
- 零配置 — 不需要 API Key，不需要写模板，不需要调 CSS
- 零门槛 — 打开就能用，一切操作都是点选、下拉、按钮，没有任何代码输入框
- 零代码 — **目标用户是外贸销售，不懂技术。界面里绝不出现 CSS 数值、HTML 标签、API 配置、正则表达式、技术参数等任何需要编码知识的输入项。一切技术细节由系统预置处理。**
- 防封禁 — 所有爬虫内置频率控制（随机延迟、UA 轮换、请求队列），确保不会被目标网站封 IP
- 聚焦发展中国家 — 中东、非洲、中亚、东南亚、南亚、拉美、东欧

## 二、用户流程（3 步）

```
┌──────────────┐     ┌──────────────────┐     ┌─────────────────┐
│ ① 添加产品    │ ──▶ │ ② 选市场 → 搜索   │ ──▶ │ ③ 客户列表+导出   │
│  中文名/英文名  │     │  区域/国家/城市    │     │  背调+开发信     │
│  品类/规格/价格 │     │  自动搜+手动补     │     │  Excel 一键导出  │
└──────────────┘     └──────────────────┘     └─────────────────┘
```

|------|------|:---:|
| 中文产品名 | 如：保温钉（外墙保温锚固钉） | ✅ |
| 英文产品名 | 如：Insulation Anchor | ✅ |
| 品类 | 如：建筑五金 | — |
| 子类目 | 如：保温锚固件 | — |
| 英文关键词 | 用于搜索，逗号分隔 | ✅ |
| 产品描述（中文） | 一段话介绍 | — |
| 产品描述（英文） | 用于生成开发信和落地页 | ✅ |
| 规格/型号 | — | — |
| 材质 | — | — |
| 报价（FOB USD） | — | — |
| 起订量（MOQ） | — | — |

### 步骤 ②：选市场 → 搜索

**地区选择**（两级联动）：
- **一级：大区域** — 中东 / 非洲 / 中亚 / 东南亚 / 南亚 / 拉美 / 东欧
- **二级：国家** — 选区域后展开该区域下的国家列表
- **可选：城市** — 选国家后可指定城市（如 Dubai, Lagos, Almaty）

**获客渠道（4 种，全部免费）**：

### 渠道 1：黄页/商业目录采集 ⭐ 主力渠道
| 黄页网站 | 覆盖区域 | 内容 |
|----------|----------|------|
| europages.com | 欧洲 | 按行业/产品分类的公司目录 |
| kompass.com | 全球 | B2B 公司数据库，支持行业+地区筛选 |
| tradekey.com | 全球 | 进出口商目录 |
| yellowpages.com.ng/ke/ae 等 | 各国本地 | 本地商家目录（按国家分站） |
| alibaba.com | 全球 | 供应商列表页公开可见 |
| exportersindia.com | 印度 | 出口商目录 |
| turkishexporter.net | 土耳其 | 土耳其出口商 |

**实现方式**：用户选择黄页来源 + 输入关键词 → 系统自动抓取页面中的公司名、电话、邮箱、网址 → 去重后导入线索库。纯 requests + BeautifulSoup，零成本。

### 渠道 2：Google 搜索结果抓取
- 绕过 SerpAPI，直接解析 Google 搜索结果页 HTML
- 轮换 User-Agent + 3-5 秒间隔，免费做到 50-100 次/天
- 自动组合搜索词：`"{产品英文关键词}" + "{国家/城市}" + "importer OR wholesaler OR distributor"`

### 渠道 3：WHOIS 域名反查
- 已知某行业关键词 → 搜索相关域名 → WHOIS 查询
- 可拿到注册人邮箱和电话，适合精准挖掘
- 作为补充渠道，用于特定赛道深挖

### 渠道 4：Google Maps 商家搜索
- Google Places API 每月 $200 免费额度（约 28,000 次查询）
- 搜索 `"plastic manufacturer in Dubai"` → 商家名、地址、电话、网站
- 可选启用，免费额度内不花钱

**统一处理**：
- 4 个渠道的结果汇入同一个线索库
- 按域名自动去重（同一公司不同渠道来源只保留一条）
- 每条线索标注来源渠道标签（黄页/Google搜索/WHOIS/Maps）

### 步骤 ③：客户列表 + 导出

**客户列表**：
| 字段 | 说明 |
|------|------|
| 公司名 | — |
| 国家/城市 | — |
| 官网 | 可点击跳转 |
| 邮箱 | 从官网自动提取 |
| 电话/WhatsApp | 从官网自动提取 |

| 可信度 | 高/中/低/未知（简单三档，不用复杂评分） |

**操作**：
- ✅ 查看公司详情（含背调信息）
- ✅ 勾选 → 生成多语言开发信
- ✅ 勾选 → 导出 Excel
- ✅ 标记已联系/已忽略

## 三、公司背调（中等程度，免费实现）

**自动抓取**（无需 API）：
1. 访问公司官网首页 + About 页面
2. 提取邮箱、电话、WhatsApp、社交媒体链接
3. 提取产品/经营范围关键词
4. 判断网站是否活跃（最近更新、是否有产品目录）

**背调评分（简化版，3 档）**：
| 可信度 | 判定标准 |
|:---:|------|
| 高 🟢 | 有官网 + 完整联系方式 + 产品匹配 |
| 中 🟡 | 有官网 + 部分联系方式 |
| 低 🔴 | 无官网或信息极少 |
| 未知 ⚪ | 未背调 |

## 四、开发信与落地页

### 开发信
- 根据产品信息 + 公司名称 + 目标国家 → 自动生成
- 语言：自动匹配目标国家的商业语言
  - 中东 → 英语 + 阿拉伯语
  - 非洲 → 英语 + 法语（西非/中非）
  - 中亚 → 俄语
  - 拉美 → 西班牙语/葡萄牙语
  - 东南亚 → 英语
- 3 种模板：初次联系 / 报价推介 / 跟进
- 支持一键复制

### 产品落地页（简化版）
- 自动生成一个简洁的单页 HTML（产品名 + 描述 + 规格 + 报价 + 联系方式）
- 语言跟随目标市场
- 下载为 HTML 文件，用户可自行托管（如 GitHub Pages、Netlify 免费版）
- **不做询盘接收功能**（简化掉之前的 HTTP 服务）

## 五、技术方案

| 项目 | 选择 | 理由 |
|------|------|------|
| 框架 | Streamlit | 纯 Python，快速开发，已有基础 |
| 数据库 | SQLite | 本地存储，零配置 |
| 前端 | Streamlit 原生组件 + 少量 CSS | 不需要用户调样式 |
| 多语言 | 内置模板 + 语言映射表 | 根据国家自动选语言 |
| 部署 | 服务器 systemd | 已有基础设施 |

**获客渠道技术实现**：

| 渠道 | 技术 | 成本 | 优先级 |
|------|------|:---:|:---:|
| 黄页采集 | requests + BeautifulSoup + lxml | 免费 | P0 ⭐ |
| Google 搜索 | HTTP GET 解析 HTML，轮换 UA，3-5s 间隔 | 免费（50-100次/天） | P0 |
| WHOIS 反查 | python-whois 库 | 免费 | P1 |
| Google Maps | Google Places API（$200/月免费额度） | 免费额度内无成本 | P1 |

**通用基础设施**：
- 统一 URL 去重（按域名）
- 统一联系方式提取（邮箱/电话/WhatsApp 正则）
- 请求频率控制（队列 + 随机延迟 2-5s + UA 轮换）
- 所有渠道结果汇入同一个线索库，标注来源标签
