# TradeLead Intel V3.0 — UI 设计方案交付报告

## 已完成工作

为 TradeLead Intel 外贸线索情报系统创建了完整的新一代 UI 设计系统，定位 **"Precision Trade 精准外贸"** 专业商务风格。

## 核心交付

**`outputs/tradelead-design-system.html`** — 综合性设计系统文件，包含三个可切换视图：

### 视图 1：仪表盘首页
- 5 个指标卡片（带左侧色条引导线）
- 最近公司线索数据表格
- 评级分布（A/B/C/D 四档芯片式展示）
- 合规风险提示
- 6 个快速操作入口

### 视图 2：公司详情页
- 综合评分面板（圆形评分 + 分项进度条）
- 背调证据摘要
- CRM 跟进记录表格
- 新增跟进表单

### 视图 3：设计规范文档
- 完整色彩系统（品牌色/语义色/评级色 + 色块预览）
- 字体层级表（Plus Jakarta Sans + JetBrains Mono）
- 8 种组件交互状态矩阵
- 4px 基础间距系统视觉化

## 设计决策

| 维度 | 选择 | 理由 |
|------|------|------|
| 主色 | Teal #0E7C7B | 区别于通用蓝色，体现国际贸易和国际视野 |
| 底色 | Warm Gray #F5F3F0 | 替代纯白，减少数据密集型页面的视觉疲劳 |
| 导航 | 深海蓝暗色侧边栏 | 专业权威感，与 Bloomberg 终端类似基调 |
| 评级 | A=绿 B=蓝 C=橙 D=红 | 直观语义，符合国际贸易风险分级习惯 |
| 字体 | Plus Jakarta Sans | 现代几何感 sans-serif，避免 Inter/Roboto 泛化 |
| 间距 | 4px base | 比 8px 更精细，适配数据密集场景 |

## 与现有设计的对比

- 侧边栏从深灰蓝 → 深海蓝（更有层次感）
- 主色从纯蓝 #2563eb → 青蓝 #0E7C7B（更独特）
- 底色从冷灰 #f8fafc → 暖灰 #F5F3F0（更亲和）
- 卡片增加左侧色条引导线（更强的视觉层次）
- 引入金色点缀用于 A 级/高价值线索（差异化标识）
